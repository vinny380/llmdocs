# Code of conduct

We want an inclusive, harassment-free experience for everyone who participates in **issues, discussions, pull requests, and chat** around this project.

## Our standards

**Encouraged:**

- Respectful, constructive feedback.
- Assuming good intent; asking clarifying questions.
- Focusing on what helps users and contributors.

**Not acceptable:**

- Harassment, discrimination, or personal attacks.
- Trolling, insulting comments, or sustained disruption.
- Publishing others’ private information without permission.

## Reporting

If you see a violation, **contact the maintainers** via GitHub (issue or direct message if available). We will review reports fairly and may take action up to **temporary or permanent ban** from project spaces for harmful behavior.

## Scope

This applies to all project spaces (repository, issues, PRs, and project-related channels the maintainers designate).
